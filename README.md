# 🚀 JobPortal - React Job Listing App

Modern, responsive job portal built with **React + Vite + Real API**. Features navbar, hero, features, job cards, apply modal, favorites, smooth scroll.


## ✨ Features
- ✅ Responsive design (mobile-first)
- ✅ Real API integration (dummyjson.com)
- ✅ Apply modal with form
- ✅ Heart favorites system
- ✅ Smooth scrolling navbar
- ✅ Loading states & error handling
- ✅ Professional UI/UX
